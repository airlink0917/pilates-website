ピラティスウェブサイトをモバイルで見る方法
========================================

方法1: Python HTTPサーバーを使用（推奨）
--------------------------------------
1. start-server.bat をダブルクリックして実行
2. コマンドプロンプトが開き、サーバーが起動します
3. 別のコマンドプロンプトを開いて「ipconfig」と入力
4. 「IPv4 アドレス」を確認（例：192.168.1.100）
5. モバイルデバイスのブラウザで以下を入力：
   http://[あなたのIPアドレス]:8000
   例: http://192.168.1.100:8000

方法2: Node.jsサーバーを使用
----------------------------
1. コマンドプロンプトを開く
2. cd C:\Users\user\pilates-website と入力
3. node server.js と入力
4. 表示されたモバイル用URLをモバイルブラウザに入力

方法3: Visual Studio Code Live Server（VSCodeをお使いの場合）
----------------------------------------------------------
1. VSCodeでプロジェクトフォルダを開く
2. index.htmlを右クリック
3. 「Open with Live Server」を選択
4. 表示されたアドレスのlocalhostを自分のIPアドレスに変更してモバイルでアクセス

重要な注意事項
--------------
- PCとモバイルデバイスは同じWi-Fiネットワークに接続されている必要があります
- Windowsファイアウォールでポート8000または3000を許可する必要がある場合があります
- プライベートネットワーク上でのみアクセス可能です（インターネットからはアクセスできません）

トラブルシューティング
--------------------
- アクセスできない場合は、Windowsファイアウォールの設定を確認してください
- アンチウイルスソフトがブロックしている可能性もあります
- IPアドレスが正しいか再度確認してください
